快速部署指南

方案选择：

  1. 纯静态部署（推荐初次使用）：上传整个文件夹到任意Web服务器即可
  2. Node.js服务器（推荐生产）：提供完整API、数据持久化、语音文件上传
  3. PHP服务器（传统企业）：适合已有PHP环境的团队
  4. 云平台部署（快速上线）：Vercel、GitHub Pages一键部署

最快启动（方案1）代码示例：

# 1. 上传文件到服务器（任何支持静态文件的Web服务器）
scp -r project/* user@your-server:/var/www/html/

# 2. 通过浏览器访问
https://your-domain.com/index.html

# 3. 立即开始使用！ ✅

生产部署（方案2）代码示例：

# 1. 启动Node.js服务器
cd server/node-api
npm install
npm start

# 2. 修改前端API配置（questionnaire.js）
const API_CONFIG = {
    baseUrl: 'http://your-server:3000/api',
    useServerAPI: true
};

# 3. 获得完整功能：数据分析、语音存储、多用户管理
