## 🔧 关键技术实现

### 动态配置系统
支持通过URL参数动态配置：
- `?email=接收邮箱@example.com` - 数据发送邮箱
- `?title=自定义标题` - 问卷标题
- `?contact=联系人名字` - 感谢页面显示的联系人
- `?voice=true/false` - 语音功能开关
- `?service=formspree` - 邮件服务选择

### 邮件集成方案
1. **Formspree集成**：免费50封/月，通过URL配置Form ID
2. **直接邮件发送**：通过后端API（需要部署服务器）
3. **本地存储备份**：自动备份到用户浏览器，可导出

### 语音录制技术
- 基于Web标准的MediaRecorder API
- 实时音频波形可视化
- 本地base64存储，可上传服务器
- 自动清理过期语音文件
