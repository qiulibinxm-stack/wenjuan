### 第二步：一键部署到Vercel（1分钟）

1. **登录** Vercel.com
2. **点击** "Add New Project"
3. **导入** 你的GitHub仓库 (wenjuan)
4. **点击** Deploy

**完成！** 系统会自动获得URL，例如：
`https://wenjuan.vercel.app`

### 第三步：设置邮箱配置（2分钟）

你的问卷系统支持**动态邮箱配置**：

#### 方案A：使用Formspree（推荐）
1. 访问 https://formspree.io 注册（免费）
2. 创建新表单，获得类似 `xqvyzjpw` 的Form ID
3. 用以下链接分享问卷：

```
https://wenjuan.vercel.app/?email=你的邮箱@example.com&form_id=xqvyzjpw
```

#### 方案B：URL参数配置（最简单的）

**分享包含邮箱的链接**：
```
https://wenjuan.vercel.app/?email=接收问卷的邮箱@你的域名.com
```

**带个性化配置的链接**：
```
https://wenjuan.vercel.app/?email=xxx@xx.com&title=张三的IP问卷&contact=李顾问
```

**参数解释**：
- `email`：数据发送到的邮箱
- `title`：问卷标题（可自定义）
- `contact`：联系人名称（显示在感谢页面）
- `service`：formspree（如果用Formspree）

## 🆘 遇到问题？

### 常见问题1：Vercel部署失败
**解决**：检查GitHub仓库是否公有（公开），私有仓库需要Vercel Pro版

### 常见问题2：收不到邮件
**解决**：
1. 检查垃圾邮件箱
2. 尝试用Formspree服务
3. 使用URL参数 `&debug=true` 查看调试信息

### 常见问题3：微信内不能录音
**解决**：
1. 确保链接是HTTPS（Vercel自动提供）
2. 引导用户用系统浏览器打开
3. 或建议用户用文字回答
